Este proyecto consiste en el desarrollo de un sistema de gestión de clientes, servicios y reservas para la empresa Software FJ, implementado en Python utilizando programación orientada a objetos.

El sistema permite simular el registro de clientes, la creación de servicios (reserva de salas, alquiler de equipos y asesorías) y la realización de reservas. Además, incluye manejo avanzado de excepciones para controlar errores sin detener la ejecución del programa, así como el registro de eventos en un archivo de log.

El objetivo principal es demostrar una aplicación robusta y estable, capaz de seguir funcionando correctamente incluso cuando se presentan errores.
